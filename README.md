## 🌐 English

### 📖 Overview
**Pokémon Infinite Gacha** is a completely free, unlimited web application where you can spin a gacha machine to catch random Pokémon. It fetches real-time data directly from [PokéAPI](https://pokeapi.co/), allowing you to enjoy endless encounters with Pokémon from all regions.

### ✨ Features
- **Infinite Free Spins:** No limits! Spin the gacha as many times as you want.
- **Region Selection:** Choose which region's Pokémon you want to catch (from Kanto to Paldea, or All Regions).
- **Authentic Pokémon Cries:** Hear the actual cry of the Pokémon you catch!
- **Retro Sound Effects:** Custom synthesized Game Boy-style 8-bit sound effects using Web Audio API.
- **Detailed Pokédex Data:** Displays official artwork, types, height, weight, abilities (including hidden ones), base stats, and flavor text.
- **Share to X (Twitter):** Automatically copies the caught Pokémon's image to your clipboard so you can easily paste and share it on X.
- **Bilingual Support:** Switch instantly between English and Japanese.
- **Responsive Design:** Fully optimized for mobile and desktop browsers.

### 🛠 Technologies Used
- HTML5 / CSS3 / JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com/) (Styling)
- [FontAwesome](https://fontawesome.com/) (Icons)
- [PokéAPI](https://pokeapi.co/) (Data source & Audio)
- Canvas Confetti (Particle effects)

### 🚀 Getting Started
Simply open the `index.html` file in any modern web browser, or host it via GitHub Pages.
No installation or build process is required!

## 🇯🇵 日本語

### 📖 概要
**ポケモン ガシャポン**は、完全無料で何度でも回せるWebブラウザ用のガチャアプリです。公式のデータを提供する [PokeAPI](https://pokeapi.co/) と直接連携しており、全地方のポケモンたちと無限に出会うことができます。

### ✨ 主な機能
- **永久無料・無限ガチャ:** 回数制限なし！好きなだけガチャを回せます。
- **地方の選択:** カントー地方からパルデア地方まで、出現させたい地方を絞り込んで回すことができます。
- **本物の「鳴き声」再生:** ポケモンをゲットした瞬間、PokeAPIから取得した本物の鳴き声が鳴ります！
- **レトロな効果音:** Web Audio APIを活用し、ゲームボーイ風の懐かしいピコピコ音（UI音、ファンファーレ）を完全再現。
- **詳細な図鑑データ:** 公式イラスト、タイプ、高さ、重さ、特性（夢特性含む）、種族値のステータスバー、最新の図鑑説明文を表示します。
- **X（旧Twitter）シェア機能:** ガチャ結果の画像をワンクリックでクリップボードにコピーし、そのままXに貼り付けて投稿できます。
- **多言語対応:** 日本語と英語をボタン一つで切り替え可能です。
- **レスポンシブデザイン:** スマホでもPCでも崩れず快適に遊べます。

### 🛠 使用技術
- HTML5 / CSS3 / JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com/) (スタイリング)
- [FontAwesome](https://fontawesome.com/) (アイコン)
- [PokéAPI](https://pokeapi.co/) (データおよび音声の取得元)
- Canvas Confetti (紙吹雪のエフェクト)

### 🚀 使い方
本リポジトリをクローンまたはダウンロードし、ブラウザで `index.html` を開くだけで遊べます。
ビルド環境などは一切不要です。（GitHub Pages等にアップロードするだけで即座に公開できます）
